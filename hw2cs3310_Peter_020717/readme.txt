This program was made in Eclipse

If you're using Eclipse, simply go to File -> Import -> General -> Existing Projects into Workspace, and select the folder after it has been extracted from the .zip file

To use the program, first enter the value of n (Number of value you want generated)
Then, enter the value of m (Range of values you want generated from in the range [1, m))
Program will print out the original array/linked list as well as the result from each sorting method for that array/linked list
Program will also print out answers to three problems
