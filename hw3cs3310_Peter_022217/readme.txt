This program was made in Eclipse

If you're using Eclipse, simply go to File -> Import -> General -> Existing Projects into Workspace, and select the folder after it has been extracted from the .zip file

To use the program, first enter the sequence length (number of operations you want executed)
Program will print out CBT, SBT Preorder, and SBT Inorder if the trees reach a size of 30
If the tree does not a reach a size of 30, nothing will be printed
