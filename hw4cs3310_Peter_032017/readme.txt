This program was made in Eclipse

If you're using Eclipse, simply go to File -> Import -> General -> Existing Projects into Workspace, and select the folder after it has been extracted from the .zip file

Program will print out data regarding 20 BST tests as well as the minimum and maximum heights of the BSTs and a preorder, postorder, and inorder traversal of the final BST.

THE EXTRA CREDIT IMPLEMENTATION OF HEAPS ON COUNTER VALUES IS USED IN THIS PROGRAM.
